INTRODUCTION

The Main objective of this research area is to identify topics that have been discussed through one particular chat session and the way it changed along the message count. Also it try to learn relationship of identified topics using the pattern created by the system.

Machine learning technologies for finding appropriate algorithms topics and clustering algorithms for categorize (group) the messages or any other algorithms will be used for this topic detection. And based on identified topics system will generate a graphical view for the purpose of showing the analytical data to the user in user friendly manner then user will be able to identify how the topics varied when the number of messages get increased with the time and what are the other factors that actually caused to make it changed. Also system will be able to give a chance to the user think about it logically with its graphical review.
